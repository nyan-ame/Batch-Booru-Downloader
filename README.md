# Batch Booru Downloader
Удобный и дружелюбный загрузчик со следующих сайтов:
* [Danbooru](https://danbooru.donmai.us/)
* [Gelbooru](https://gelbooru.com/)
* [Sankaku](https://chan.sankakucomplex.com/)
* Konachan ([SFW](https://konachan.net), [NSFW](https://konachan.com/))
* [yande.re](https://yande.re/url)
* [pixiv](https://pixiv.net)
* [Twitter](x.com)

## Известные проблемы
* Cookies на Danbooru живут всего 30 минут. Это ограничение сайта и обойти его нельзя.

## Инструкция по работе
0) Убедитесь, что у вас установлен [Python](https://www.python.org/downloads/)
1) Скачайте и распакуйте программу в любое удобное место. Откройте командную строку в папке с файлом `main.py` (Нажать на адресную строку в проводнике, написать cmd и нажать Enter)
2) В командной строке напишите `pip install -r requirements.txt`, дождитесь установки необходимых библиотек.
3) Откройте `settings.env`. Укажите все данные для тех сайтов, с которых Вы собираетесь загружать. Для сайтов, откуда вы ничего загружать не собираетесь, оставляете переменные пустыми (после `=` ничего писать НЕ НУЖНО!)
   > Все параметры, куда не указываются данные для авторизации, стоят ровно так, как рекомендуется.
4) Запустите `main.py`. Стрелками выберите сайт, откуда хотите начать загрузку. Управление работает только на английской раскладке и написано снизу, а также осуществляется мышью, если вы почему-то этого захотели. Переключение между вкладками осуществляется цифрами 1 и 2.
   > Для danbooru каждый раз нужно обязательно выдавать cookies. [Подробнее здесь]([url](https://github.com/nyan-ame/Batch-Booru-Downloader/blob/main/DanbooruCookies.md)).
   
   > Для pixiv единожды нужно будет установить chrome через playwright. Команда `playwright install chrome`.
5) Скрипт загружает файлы в `.../Batch Booru Downloader/downloads/`. После окончания загрузки скрипт можно закрыть.

**Скрипт не ворует ваши api ключи и логины/пароли!!!**
> Для особо ленивых и суеверных - вы всегда можете отправить исходный код на изучение любой нейросети.
